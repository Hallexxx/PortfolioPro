Le site web a été créé en Python en utilisant le framework Django pour offrir une plateforme de gestion des employés.
Il utilise GoogleCloud pour le stockage des données en ligne, assurant ainsi une gestion efficace et sécurisée des informations.
Le site comprend un système d'authentification permettant aux utilisateurs de se connecter, de s'inscrire et d'accéder à des fonctionnalités d'administration.
Parmi ces fonctionnalités, on trouve les opérations CRUD (Create, Read, Update, Delete) qui permettent de gérer les informations des employés.
De plus, un tableau de bord administratif a été mis en place pour faciliter la modification, l'ajout et la suppression des données des employés, rendant la gestion interne plus fluide et organisée.

Ce projet consiste en la réalisation d'un Pokedex interactif utilisant React. Projet Scolaire Symfony
Bibliothèque Musicale

API sécurisée avec JWT pour gérer artistes, albums et chansons. Permet CRUD, filtres et gestion de favoris...

Cette API développée avec Symfony et API Platform gère les artistes, albums et chansons d'une bibliothèque musicale. Elle permet aux utilisateurs authentifiés d'ajouter des éléments en favoris et offre un CRUD complet pour les administrateurs. La sécurisation est assurée via un système de tokens JWT, et les données peuvent être filtrées selon des critères comme le style musical ou la durée des chansons. Les relations entre les ressources sont respectées et des endpoints avancés sont disponibles pour accéder à des sous-ressources. Une base SQLite est incluse pour tester l'application.

Projet Scolaire Node.js
Site Micro-crèches

Application web pour gérer plannings et utilisateurs d’une micro-crèche avec Angular et Laravel...

Ce site est conçu pour la gestion des plannings dans une micro-crèche. Développé avec Angular pour le frontend et Laravel pour le backend, il permet d'ajouter, modifier et supprimer (CRUD) des employés et des enfants inscrits. Les horaires sont automatiquement affectés au planning selon les jours de garde des enfants et les horaires des employés, en respectant un ratio de 1 employé pour 3 enfants. La gestion des accès est strictement définie : seul le gérant peut effectuer des modifications, tandis que les employés peuvent consulter leur profil et le planning. L’application inclut un support multilingue (français et anglais), avec des filtres pour basculer entre les langues.
